## üé¨ Demo Walkthrough

### **SALES Role Demo Walkthrough**

#### Step 1: Login
```
Username: sales
Password: sales123
```

#### Step 2: Explore Dashboard
- View overall fleet statistics
- See active vehicles count (25,000+)
- Check sales metrics and environmental impact

#### Step 3: Add New Vehicle
1. Navigate to **"Add Vehicle"** from sidebar
2. Fill in vehicle details:
   - Vehicle Type: Electric Scooter / tri Wheeler / MPV
   - VIN Number (use QR scanner or manual entry)
   - Model Selection
   - VCU Number
   - Battery Serial Number
   - Battery Capacity
   - IMEI Number
   - Customer Details (optional)
3. Click **"Save"** to add vehicle to fleet

#### Step 4: Manage Dealers
1. Navigate to **"Dealer Management"**
2. View all dealer profiles:
   - Mumbai Motors (D001)
   - Delhi Auto Hub (D002)
   - Bangalore EV Center (D003)
3. Click **"Add New Dealer"** to register new dealership
4. View dealer inventory (2-wheelers and 3-wheelers)
5. Check vehicle status (in-stock vs assigned)

#### Step 5: Vehicle Insights
1. Navigate to **"Vehicle Insights"**
2. Select any vehicle from the list
3. View comprehensive metrics:
   - Total trips and distance
   - Average speed and efficiency
   - Maintenance costs
   - Revenue generated
   - Health scores
   - Monthly trends

#### Step 6: Generate Reports
1. Navigate to **"Reports"**
2. Select report type:
   - Vehicle Performance
   - Fleet Analytics
   - Dealer Reports
   - Environmental Impact
3. Apply filters (date range, vehicle, dealer)
4. Generate and download reports (PDF/Excel)

#### Step 7: Customer Support
1. Navigate to **"Support & Connect"**
2. Submit customer complaints
3. Track complaint status
4. View support dashboard

## üìù Notes

### Important Points:

1. **Password Security**: All demo passwords are simple for testing. In production, enforce strong password policies.

2. **Data Filtering**: 
   - SALES role sees ALL data (no filtering)
   - FLEET_DRIVER role sees ONLY assigned vehicle data (strict filtering)

3. **Assigned Vehicles**:
   - `driver` ‚Üí V000001
   - `driver_user` ‚Üí V000002
   - `fleet_driver` ‚Üí V000003

4. **Role Switching**: To test different roles, logout and login with different credentials.

5. **Browser Storage**: Login state is stored in localStorage. Clear browser data to reset.

## üîß Troubleshooting

### Common Issues:

**Issue**: "Invalid username" error
- **Solution**: Ensure username is typed correctly (case-insensitive)

**Issue**: "Invalid password" error
- **Solution**: Check password matches exactly (case-sensitive)

**Issue**: Menu items not showing
- **Solution**: Role-based access is working correctly. Some features are restricted by role.

**Issue**: Cannot see all vehicles
- **Solution**: FLEET_DRIVER role is designed to see only assigned vehicles. This is expected behavior.

**Issue**: Cannot add vehicles
- **Solution**: FLEET_DRIVER role doesn't have permission to add vehicles. Use SALES or DEALER role instead.
